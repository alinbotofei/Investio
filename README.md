# Investio

React + Vite + Tailwind frontend (frontend/)

Requirements

- Node.js 18+
- npm

Quick start (from project root)

```bash
npm install
npm run dev
```

Build

```bash
npm run build
npm run preview
```

Notes

- Vite root: `frontend/`
- Frontend entry: `frontend/src/main.tsx`
- Run commands from the project root
- Project layout follows Atomic Design: `atoms/`, `molecules/`, `organisms/`, `templates/`, `pages/`
